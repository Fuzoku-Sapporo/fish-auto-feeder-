# 🐠 自動給餌システム

マイクロビットを使った魚の自動給餌システムです。
定時給餌と手動給餌の両方に対応しており、手動給餌で12時間タイマーがリセットされます。

## 機能

### マイクロビット側
- **ボタンA**: 手動給餌（タイマーリセット）
- **ボタンB**: 次回給餌まで何時間か表示
- **Bluetooth**: パソコンからのコマンド受信で給餌
- **自動実行**: 12時間ごとに自動給餌

### パソコン側
- **定時給餌**: 12時間ごとに自動給餌
- **手動給餌**: いつでも給餌コマンド送信可能
- **タイマーリセット**: 手動給餌で12時間カウンターをリセット
- **ステータス表示**: 次回給餌までの時間を表示

## ハードウェア

- マイクロビット
- サーボモーター（P16ピン接続）
- 給餌装置（ホッパー/リール）

## セットアップ

### 1. マイクロビットにプログラムをアップロード

1. MakeCode を開く: https://makecode.microbit.org
2. `microbit_feeder.js` をコピー
3. JavaScript エディタに貼り付け
4. ダウンロード → マイクロビットに転送

### 2. パソコンに環境構築

```bash
# ライブラリをインストール
pip install bleak schedule

# プログラムを実行
python feeder_12hour.py
```

### 3. Bluetooth ペアリング

- Windows: 設定 → Bluetooth とデバイス → マイクロビットをペアリング
- Mac: システム設定 → Bluetooth → マイクロビットをペアリング

## 使い方

### マイクロビット
| ボタン | 動作 |
|--------|------|
| **A** | 🐟 手動給餌 |
| **B** | ⏱️ 次回給餌時間表示 |

### パソコン
プログラム実行後、以下のコマンドが使用可能：
- `feed` - 今すぐ給餌
- `status` - 給餌ステータス確認
- `interval` - 給餌間隔を変更（例：8時間、24時間など）
- `exit` - 終了

## ファイル構成

```
fish-auto-feeder-/

├── feeding_config.json     # 設定ファイル
├── last_feed.json         # 給餌履歴
└── README.md              # このファイル
```

## トラブルシューティング

### マイクロビットが見つからない
- マイクロビットが電源オンか確認
- Bluetooth が ON か確認
- ペアリング済みか確認

### Bluetooth 接続エラー
```bash
pip install --upgrade bleak
```

### ポート指定エラー
マイクロビットのアドレスを確認して設定ファイルに記載
